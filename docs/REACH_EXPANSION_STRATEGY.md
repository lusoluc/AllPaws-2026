# Reichweiten-Erweiterung: AllPaws-2026 🐾

Da wir den Fokus aktuell rein auf die Erhöhung der Sichtbarkeit (Reichweite) legen wollen, konzentrieren wir uns auf Plattformen, die organisch ein hohes Grundrauschen und viele Besucher haben. 

Hier ist der Plan, wie wir AllPaws-2026 in globale Verzeichnisse und Entwickler-Feeds eintragen.

---

## 🗺️ Die Reichweiten-Kanäle

1. **AlternativeTo.net** (Der stärkste organische Kanal für Software-Suchende)
2. **Hacker News („Show HN“)** (Für globale Tech-Sichtbarkeit unter Entwicklern)
3. **GitHub Awesome-Lists** (Für dauerhaften Traffic aus der Self-Hosting-Szene)
4. **Developer-Newsletter (Direkt-Einreichung)** (Kurze Online-Formulare ohne Mail-Pitching)

---

## 🛠️ Kanal 1: AlternativeTo.net (Sehr empfohlen)

Menschen, die unzufrieden mit teurer Tierheim-Software (SaaS) sind, suchen bei Google nach *„Alternative to Shelterluv“* oder *„Alternative to Animal Shelter Manager“*. AlternativeTo.net ist bei diesen Suchanfragen fast immer Platz 1.

### Klick-Anleitung für dich:
1. Gehe auf [alternativeto.net](https://alternativeto.net).
2. Erstelle ein kostenloses Konto (oder logge dich über GitHub ein).
3. Klicke ganz unten im Fußbereich der Seite auf **„Suggest an app“** (App vorschlagen).
4. Fülle die Felder aus:
   * **App Name:** `AllPaws-2026`
   * **Website URL:** `https://github.com/lusoluc/AllPaws-2026`
   * **License:** `Open Source (MIT)`
   * **Platforms:** `Web, Self-Hosted`
   * **Short description:** *„A free, offline-first mobile-friendly web companion for animal shelters and rescue groups to replace spreadsheets.“*
   * **Alternatives to:** Trage dort ein: `Shelterluv`, `Animal Shelter Manager`.

---

## ✍️ Kanal 2: Hacker News („Show HN“)

Hacker News ist das größte Entwickler-Forum der Welt. Ein erfolgreicher Post dort kann Zehntausende Besucher an einem Tag bringen. Der Beitrag muss im Format **„Show HN“** verfasst sein.

### Der Text-Entwurf (Copy & Paste):
**Titel (Title):**
> Show HN: AllPaws-2026 – Offline-first web companion for animal shelters

**Text (Casual & Tech-fokussiert):**
```text
Hi HN,

I built AllPaws-2026 because local animal shelters are drowning in administrative work (and spreadsheets) when they should be caring for animals.

It's a mobile-first web companion built with Next.js, Dexie/IndexedDB, and Supabase. 

Key technical highlights:
- 100% Offline-First: Syncs via background queues when Wi-Fi is available. Essential for field rescues in dead zones.
- Browser-based Audio Stitching: Uses the Web Audio API to let volunteers stitch multiple voice descriptions together directly on their mobile browsers.
- Social Postcard Generator: Renders custom shareable graphic cards from database profiles using HTML5 Canvas.

It's MIT-licensed and has a full Jest test suite (15 suites, 99 tests) validating the sync conflict resolution and audio stitching logic.

I’d love to hear your feedback on the offline synchronization design!

Repo: https://github.com/lusoluc/AllPaws-2026
```

## 📂 Kanal 3: GitHub „Awesome Lists“

Es gibt das weltberühmte Repository `awesome-selfhosted`. Es ist die Bibel für jeden, der eigene Server betreibt. Wenn wir dort gelistet sind, bekommen wir wöchentlich neuen Traffic.

### Klick-Anleitung für dich:
1. Gehe auf das Repository [github.com/awesome-selfhosted/awesome-selfhosted](https://github.com/awesome-selfhosted/awesome-selfhosted).
2. Klicke oben rechts auf **„Fork“**, um eine Kopie in deinem Account zu erstellen.
3. Bearbeite in deinem Fork die Datei `README.md` und füge uns unter der Kategorie **„Animal Shelter Management“** (oder ähnlich) hinzu:
   * *Format:* `* [AllPaws-2026](https://github.com/lusoluc/AllPaws-2026) - Offline-first mobile web companion for animal shelters. (MIT, Nodejs)`
4. Erstelle einen **Pull Request** zurück zum Haupt-Repository.
